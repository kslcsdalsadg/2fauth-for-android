FAuth
by Bubka

Info

2FAuth is a web based self-hosted alternative to One Time Passcode (OTP) generators like Google Authenticator, designed for both mobile and desktop.

Bubka's 2FAuth is the core of our app, which we connect to in order to read 2FA accounts.

You will need to install this app on your computer (instructions are provided for installing it in a docker container) in order to use it.
